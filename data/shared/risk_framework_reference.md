# Risk Framework Reference
## Standard Risk Assessment Methodology

### Risk Scoring Matrix

Failure Rate (probability of adverse event per year):
- Critical: >0.3
- High: 0.15–0.3
- Medium: 0.05–0.15
- Low: 0.01–0.05
- Negligible: <0.01

Uncertainty Score:
- High (0.7–1.0): Limited data, wide confidence intervals, unverified sources
- Medium (0.4–0.7): Partial data, some verification, known gaps
- Low (0.0–0.4): Well-characterised, multiple verified sources, recent data

### Loss Range Estimation
Loss ranges should capture 10th–90th percentile outcomes:
- Direct costs: repair, remediation, regulatory fines
- Indirect costs: production downtime, reputational damage
- Third-party liability: property, environmental, human harm

### Gap Classification
Knowledge gaps should be classified by:
1. Data recency (when was last measurement/survey)
2. Data independence (operator-reported vs. third-party verified)
3. Completeness (are all relevant sub-components assessed)
4. Methodology (is the measurement approach appropriate)

### Token Factory Framework
Reducing uncertainty requires information tokens. The relationship is not linear:
- First reduction (High -> Medium): typically 1,000–5,000 tokens of analysis
- Second reduction (Medium -> Low): typically 10,000–50,000 tokens
- Full characterisation: 50,000+ tokens, often involving primary data collection
